# 🔧 Environment Variables Configuration

This document contains all the environment variables needed for production deployment.

## 🖥️ Backend Environment Variables (Render)

Copy these to your Render Web Service environment variables:

```bash
# Django Configuration
DJANGO_SETTINGS_MODULE=rbcomputer.settings_production
DEBUG=False
SECRET_KEY=[Auto-generated by Render - don't set manually]

# Database
DATABASE_URL=[Auto-populated from PostgreSQL service]

# Frontend Configuration
FRONTEND_URL=https://rbcomputer.netlify.app

# Email Configuration (SendGrid)
SENDGRID_API_KEY=[Your SendGrid API Key]
DEFAULT_FROM_EMAIL=R.B Computer <noreply@rbcomputer.com>

# Optional: Custom Domain
# FRONTEND_URL=https://yourdomain.com
```

## 🌐 Frontend Environment Variables (Netlify)

Copy these to your Netlify site environment variables:

```bash
# API Configuration
REACT_APP_API_BASE_URL=https://rbcomputer-api.onrender.com/api
REACT_APP_API_TIMEOUT=30000

# Application Configuration
REACT_APP_APP_NAME=R.B Computer
REACT_APP_APP_VERSION=1.0.0
REACT_APP_ENVIRONMENT=production

# Feature Flags
REACT_APP_ENABLE_ANALYTICS=true
REACT_APP_ENABLE_ERROR_REPORTING=true
REACT_APP_ENABLE_PERFORMANCE_MONITORING=true

# Build Configuration
NODE_VERSION=18.17.0
NPM_VERSION=9.6.7
GENERATE_SOURCEMAP=true
INLINE_RUNTIME_CHUNK=false

# Optional: Analytics and Monitoring
# REACT_APP_GOOGLE_ANALYTICS_ID=[Your GA4 ID]
# REACT_APP_SENTRY_DSN=[Your Sentry DSN]
```

## 📧 SendGrid Configuration

### Step 1: Create SendGrid Account
1. Go to [sendgrid.com](https://sendgrid.com)
2. Sign up for free account (100 emails/day)
3. Verify your email address

### Step 2: Create API Key
1. Go to Settings → API Keys
2. Click "Create API Key"
3. Choose "Full Access" (or create restricted key with Mail Send permissions)
4. Copy the API key (you won't see it again!)

### Step 3: Verify Sender
1. Go to Settings → Sender Authentication
2. Choose "Single Sender Verification" (easiest for getting started)
3. Add your email address (e.g., admin@rbcomputer.com)
4. Verify the email address

### Step 4: Add to Render
1. Go to your Render web service
2. Add environment variable:
   - **Key**: `SENDGRID_API_KEY`
   - **Value**: [Your SendGrid API Key]

## 🔐 Security Notes

### Secret Key
- **Never** set SECRET_KEY manually in Render
- Let Render auto-generate it for security
- It will be automatically created when you deploy

### API Keys
- **Never** commit API keys to Git
- Use environment variables only
- Rotate keys regularly for security

### Database URL
- **Never** set DATABASE_URL manually
- It's automatically populated when you connect the PostgreSQL service
- Use the "Internal Database URL" for better performance

## 🆘 Troubleshooting

### Common Issues:

1. **"SECRET_KEY not set" error**:
   - Don't set SECRET_KEY manually
   - Let Render auto-generate it

2. **Database connection failed**:
   - Ensure PostgreSQL service is running
   - Check DATABASE_URL is auto-populated
   - Verify both services are in same region

3. **CORS errors**:
   - Verify FRONTEND_URL matches your Netlify domain
   - Check REACT_APP_API_BASE_URL is correct
   - Ensure both use HTTPS in production

4. **Email not sending**:
   - Verify SENDGRID_API_KEY is correct
   - Check sender email is verified in SendGrid
   - Review SendGrid activity logs

5. **Build failures**:
   - Check all required environment variables are set
   - Verify NODE_VERSION and NPM_VERSION compatibility
   - Review build logs for specific errors
