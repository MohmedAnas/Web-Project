# 🚀 RBComputer Deployment Guide

Complete guide for deploying RBComputer Student Management System to production using Netlify (Frontend) + Render (Backend).

## 📋 Prerequisites

- GitHub account
- Netlify account (free)
- Render account (free)
- SendGrid account (free tier)
- Domain name (optional)

## 🏗️ Architecture Overview

```
Internet → Netlify (React App) → Render (Django API) → PostgreSQL (Render)
```

## 📦 Phase 1: Backend Deployment (Render)

### Step 1: Prepare Repository

1. **Push code to GitHub:**
   ```bash
   cd /home/master73/RBComputer
   git add .
   git commit -m "Add production configuration"
   git push origin main
   ```

### Step 2: Create Render Account

1. Go to [render.com](https://render.com)
2. Sign up with GitHub account
3. Connect your GitHub repository

### Step 3: Deploy Backend Service

1. **Create Web Service:**
   - Click "New +" → "Web Service"
   - Connect GitHub repository: `RBComputer`
   - Configure service:
     - **Name**: `rbcomputer-api`
     - **Region**: Oregon (US West)
     - **Branch**: `main`
     - **Root Directory**: `backend`
     - **Runtime**: Python 3
     - **Build Command**: `./build.sh`
     - **Start Command**: `gunicorn rbcomputer.wsgi:application --bind 0.0.0.0:$PORT --workers 2 --timeout 120`

2. **Set Environment Variables:**
   ```
   DJANGO_SETTINGS_MODULE=rbcomputer.settings_production
   DEBUG=False
   SECRET_KEY=[Auto-generated by Render]
   FRONTEND_URL=https://rbcomputer.netlify.app
   DEFAULT_FROM_EMAIL=R.B Computer <noreply@rbcomputer.com>
   SENDGRID_API_KEY=[Your SendGrid API Key]
   ```

### Step 4: Create PostgreSQL Database

1. **Create Database:**
   - Click "New +" → "PostgreSQL"
   - Configure database:
     - **Name**: `rbcomputer-db`
     - **Database Name**: `rbcomputer`
     - **User**: `rbcomputer_user`
     - **Region**: Oregon (US West)
     - **Plan**: Free

2. **Connect Database:**
   - Copy the "Internal Database URL"
   - Add to web service environment variables:
     ```
     DATABASE_URL=[Internal Database URL from PostgreSQL service]
     ```

### Step 5: Deploy and Verify

1. **Deploy Service:**
   - Click "Deploy Latest Commit"
   - Monitor build logs
   - Wait for deployment to complete

2. **Verify Deployment:**
   - Visit: `https://rbcomputer-api.onrender.com/api/health/`
   - Should return: `{"status": "healthy", ...}`

## 🌐 Phase 2: Frontend Deployment (Netlify)

### Step 1: Create Netlify Account

1. Go to [netlify.com](https://netlify.com)
2. Sign up with GitHub account
3. Connect your GitHub repository

### Step 2: Deploy Frontend

1. **Create New Site:**
   - Click "Add new site" → "Import an existing project"
   - Choose GitHub and select `RBComputer` repository
   - Configure build settings:
     - **Base directory**: `frontend`
     - **Build command**: `npm run build`
     - **Publish directory**: `frontend/build`

2. **Set Environment Variables:**
   Go to Site Settings → Environment Variables:
   ```
   REACT_APP_API_BASE_URL=https://rbcomputer-api.onrender.com/api
   REACT_APP_API_TIMEOUT=30000
   REACT_APP_APP_NAME=R.B Computer
   REACT_APP_ENVIRONMENT=production
   REACT_APP_ENABLE_ANALYTICS=true
   NODE_VERSION=18.17.0
   NPM_VERSION=9.6.7
   ```

3. **Configure Domain:**
   - Go to Site Settings → Domain Management
   - Change site name to: `rbcomputer` (if available)
   - Your site will be: `https://rbcomputer.netlify.app`

### Step 3: Deploy and Verify

1. **Deploy Site:**
   - Click "Deploy site"
   - Monitor build logs
   - Wait for deployment to complete

2. **Verify Deployment:**
   - Visit: `https://rbcomputer.netlify.app`
   - Test login functionality
   - Verify API connectivity

## 📧 Phase 3: Email Configuration (SendGrid)

### Step 1: Create SendGrid Account

1. Go to [sendgrid.com](https://sendgrid.com)
2. Sign up for free account (100 emails/day)
3. Verify your account

### Step 2: Configure SendGrid

1. **Create API Key:**
   - Go to Settings → API Keys
   - Create new API key with "Full Access"
   - Copy the API key

2. **Add to Render Environment:**
   - Go to your Render web service
   - Add environment variable:
     ```
     SENDGRID_API_KEY=[Your SendGrid API Key]
     ```

3. **Verify Email Domain (Optional):**
   - Go to Settings → Sender Authentication
   - Verify your domain or single sender email

## 🔧 Phase 4: Final Configuration

### Step 1: Update CORS Settings

1. **Update Backend CORS:**
   - Ensure your Netlify URL is in `CORS_ALLOWED_ORIGINS`
   - Redeploy backend if needed

### Step 2: Test Complete System

1. **Test User Flows:**
   - Student registration
   - Admin login
   - Course enrollment
   - Fee payment
   - Attendance marking
   - Certificate generation
   - Email notifications

### Step 3: Create Admin User

1. **Access Django Admin:**
   - Visit: `https://rbcomputer-api.onrender.com/admin/`
   - Login with: `admin@rbcomputer.com` / `admin123`
   - Create additional admin users as needed

## 🚨 Troubleshooting

### Common Issues

1. **Build Failures:**
   - Check build logs in Render/Netlify
   - Verify environment variables
   - Check dependency versions

2. **Database Connection Issues:**
   - Verify DATABASE_URL environment variable
   - Check PostgreSQL service status
   - Review connection settings

3. **CORS Errors:**
   - Verify CORS_ALLOWED_ORIGINS includes Netlify URL
   - Check frontend API base URL
   - Ensure HTTPS is used

4. **Email Issues:**
   - Verify SendGrid API key
   - Check sender authentication
   - Review email templates
