# Render Deployment Guide for R.B Computer System

## The Pip Upgrade Issue Solution

The "subprocess exit, there is an error in pip" issue on Render is caused by an outdated pip version. Here's how to fix it:

## Step 1: Updated Build Configuration

### Option A: Using Build Script (Recommended)

**Build Command in Render Dashboard:**
```bash
cd backend && chmod +x build.sh && ./build.sh
```

**Start Command:**
```bash
cd backend && gunicorn rbcomputer.wsgi:application
```

### Option B: Direct Build Command

**Build Command in Render Dashboard:**
```bash
cd backend && python -m pip install --upgrade pip && pip install -r requirements-render.txt && python manage.py collectstatic --no-input && python manage.py migrate
```

**Start Command:**
```bash
cd backend && gunicorn rbcomputer.wsgi:application
```

## Step 2: Environment Variables

Set these in your Render service dashboard:

```
DEBUG=False
```

## Step 4: Render Service Configuration

### Backend Service Setup:

1. **Service Type:** Web Service
2. **Environment:** Python 3
3. **Region:** Choose closest to your users

5. **Root Directory:** Leave empty (we use `cd backend` in commands)
6. **Build Command:**

   ```bash
   cd backend && python -m pip install --upgrade pip && pip install -r requirements-render.txt && python manage.py collectstatic --no-input && python manage.py migrate
   ```
7. **Start Command:**


2. **Database Name:** rbcomputer_db
3. **Database User:** rbcomputer_user

## Troubleshooting Common Issues

### 1. Pip Installation Errors
**Solution:** Ensure pip upgrade is the first command:
```bash
python -m pip install --upgrade pip
```

### 2. Static Files Not Loading
**Solution:** Check WhiteNoise configuration in settings.py:
```python
MIDDLEWARE = [
    'whitenoise.middleware.WhiteNoiseMiddleware',
    # ... other middleware
]
```

### 3. Database Connection Issues
**Solution:** Verify DATABASE_URL environment variable is set correctly.

### 4. CORS Issues
**Solution:** Add your frontend domain to CORS_ALLOWED_ORIGINS in settings.py.

### 5. Build Timeout
**Solution:** Optimize requirements.txt and remove unnecessary packages.
